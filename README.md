# SENG400-ASSIGMENT1
LLM-based software to support uploads, summaries, Q&amp;A and downloads


List of requiremnts for the project
Functional
1.	 User can upload notes (PDF, DOCX, TXT)  -- can we also incorporate pasting texts?
2.	 User can view uploaded documents list
3.	 User can select documents for processing
4.	 System uses LLM to generate summary
5.	 System uses LLM to generate practice questions and answers
6.	 User can choose number and difficulty of questions
7.	 UI displays summary and Q/A results
8.	 User can download outputs as TXT and CSV
9.  System stores generation history
10. User can delete uploaded files and outputs
11.	FR-12: System provides clear error handling ie gives feeback such as file type not accepted

    
   Non-Functional Requirements
   Privacy and transparent LLM data usage
    Performance target for summary generation
    Reliability with retry handling
    Simple and intuitive UI
     Secure file validation and upload protection
     Maintainable modular architecture
     Testable with unit and integration tests
     Basic accessibility compliance
